# JavaScript Calculator

A simple and responsive calculator application developed using HTML, CSS, JavaScript, and Bootstrap.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear display
- Delete last input
- Responsive user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## Learning Outcomes
- DOM Manipulation
- JavaScript Functions
- Event Handling
- User Interface Design
